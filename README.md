# yoru-shell

Сердце YoruOS. Шелл, который ставится пользователю при входе.

Базируется на форке **Caelestia** (которая сама поверх **Quickshell**).

## Этапы

- **Phase 1** — форк Caelestia, изучение архитектуры, первый Dev Preview.
- **Phase 3** — переработка Top Bar, Launcher, Dashboard, Notifications.
- **Phase 4** — Dynamic Island, AOD Lock Screen, Workspace Overview и другие killer-фичи.

## Стек

- Quickshell (runtime композитинга и IPC)
- QML (UI)
- Hyprland (композитор, на котором всё это работает)

## Статус

Пусто. Phase 0. Скоро начнём.
